Question 1: List any 6 features of 8086 microprocessor (4 marks)

Detail:
The 8086 microprocessor, developed by Intel, is a 16-bit processor that revolutionized computing in the late 1970s. It features a 20-bit address bus, enabling it to address 1 MB of memory. The architecture includes a powerful set of instructions, supports segmented memory for better management, and incorporates an instruction queue for faster processing. It also features separate data and address lines, enhancing performance and multitasking capabilities, and supports both minimum and maximum mode operations for flexible system configurations.

Question 2: Explain the function of following pins of 8085 processor (4 marks)

Detail:
i. ALE (Address Latch Enable): The ALE pin is used to demultiplex the address-data bus. It signals when the address bus is carrying valid address information, allowing the connected latch to capture and hold this address for memory operations.

ii. HOLD: The HOLD pin is used by external devices to gain control of the bus. When an external device wants to take over the system bus, it sends a HOLD signal to the processor. The processor completes its current cycle and then grants the bus by asserting the HLDA (Hold Acknowledge) signal.

iii. HLDA (Hold Acknowledge): The HLDA pin indicates that the processor has received a HOLD request and has relinquished control of the bus. The external device can then control the bus until the HOLD signal is deasserted.

Question 3: Given SS: 3450H; SP: 2A56H, calculate the physical address. (4 marks)

Detail:
To calculate the physical address in the 8086 microprocessor, we use the formula: Physical Address = Segment Address * 10H + Offset. Here, the Stack Segment (SS) is 3450H and the Stack Pointer (SP) is 2A56H.

1. Convert SS to a 20-bit segment address: 3450H * 10H = 34500H.
2. Add the SP offset: 34500H + 2A56H = 36F56H.

Thus, the physical address is 36F56H.

Question 4: List any 2 addressing modes with example. Identify the addressing mode of the instruction MOV AL, [SI] (4 marks)

Detail:
Addressing Modes:

1. Immediate Addressing Mode: The operand is specified directly in the instruction. Example: MOV AL, 34H (moves the immediate value 34H into AL).
2. Direct Addressing Mode: The effective address of the operand is given directly in the instruction. Example: MOV AX, [1234H] (moves the contents of memory location 1234H into AX).

MOV AL, [SI]: This instruction uses the Register Indirect Addressing Mode. Here, the address of the operand is specified in the SI register.

Question 5: Draw the internal architecture of 8086. (4 marks)

Detail:
The internal architecture of the 8086 microprocessor includes several key components:

- Bus Interface Unit (BIU): Handles all data and address transfers on the buses. It includes the instruction queue, segment registers, and address adder.
- Execution Unit (EU): Executes instructions from the instruction queue. It includes the ALU, general-purpose registers, and the control unit.
- Segment Registers: CS, DS, SS, ES for segment addressing.
- Instruction Queue: Prefetches up to six bytes of instructions for faster execution.
- Control Unit: Manages the execution of instructions.

Question 6: Explain memory segmentation in 8086. (4 marks)

Detail:
Memory segmentation in the 8086 microprocessor divides the memory into segments to enhance the efficiency and flexibility of data access. Each segment can be up to 64KB, and there are four main types of segments: Code Segment (CS), Data Segment (DS), Stack Segment (SS), and Extra Segment (ES). Segments are addressed using segment registers and offsets, which are combined to form a physical address. This approach allows for modular program design, easier memory management, and efficient utilization of memory.

Question 7: Difference between minimum mode and maximum mode. (4 marks)

Detail:
Minimum Mode:

  • The 8086 operates as a single processor.
  • Control signals for memory and I/O operations are generated internally.
  • Simple system design, ideal for single-processor systems.

Maximum Mode:

  • The 8086 can function as one of several processors in a multiprocessor system.
  • Requires an external bus controller (8288) to generate control signals.
  • Supports complex, high-performance computing environments with multiple processors.

Question 8: Draw the flag register of 8086. Explain any 2 control flags. (4 marks)

Detail:
The 8086 flag register consists of 16 bits and includes status, control, and system flags. Key control flags include:

1. Direction Flag (DF): Determines the direction of string operations. When set, string operations process data from higher to lower memory addresses.

2. Interrupt Flag (IF): Controls the enabling and disabling of interrupts. When set, the processor responds to maskable interrupts; when cleared, it ignores them.
